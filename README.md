# SUVIDHA – Smart Urban Virtual Interactive Digital Helpdesk Assistant

SUVIDHA is a **secure, multilingual, touch‑based kiosk web application** designed to provide citizens with unified access to civic utility services such as **Electricity, Water, Gas, and Municipal services** through a single self‑service platform.

The system is designed for **public kiosk deployment**, focusing on accessibility, simplicity, and security, especially for elderly and first‑time digital users.

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- React Router
- Context API
- Tailwind CSS / CSS Modules
- react‑i18next (multilingual support)

### Security & Architecture
- OTP‑based authentication (prototype)
- OAuth2 + JWT (architecture‑level)
- Session‑based access
- Kiosk‑mode compatible UI

### Deployment
- Vercel (Web‑based kiosk deployment)
